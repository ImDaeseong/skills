## Not yet built

Planning, manufacturing, and literal ERP/SCM/CRM *software-system integration* (connecting to and operating SAP, NetSuite, Salesforce, or a dedicated supply-chain platform, as opposed to the operations-advisory work `biz-ops` covers) specialist skills do not exist yet. When a request needs one of these, `genie` says so honestly (per its own SKILL.md) instead of improvising. To add one: research real GitHub candidates for that domain (ranked by stars, per `../_shared/CORE-LAWS.md` LAW 1), fuse the methodology into a new `SKILL.md` under a new folder, then add a row here. **Financial operations was evaluated (2026-07-18, sourced from founder cash-flow/burn-discipline content) and explicitly deferred** — the GitHub candidates found (`anthropics/claude-cookbooks`, `anthropics/financial-services-plugins`) are DCF/LBO/comps valuation tooling for finance-industry analysts, not founder cash-runway operating discipline; no dominant OSS match exists for the actual niche (confirmed via search — commercial SaaS like Runway/Pry/Puzzle dominates it instead). Do not build this from the podcast narrative alone if revisited — find a real methodology match first. **Planning (general PMP-style WBS activity-list/attribute generation, acceptance criteria) was also evaluated (2026-07-18, sourced from PM-tutorial content) and explicitly deferred** — the closest GitHub candidates (`eyaltoledano/claude-task-master`, 27,865★, MIT+Commons Clause rider; `sdi2200262/agentic-project-management`, 2,359★, MPL-2.0) are both scoped to software-dev task management for coding agents, not general PMP-style project planning, and neither carries a clean permissive license. No dominant OSS match exists for the general-PM niche itself. **"Money-making skills/business" (general side-hustle/AI-income playbooks) was evaluated (2026-07-18) and explicitly deferred** — the GitHub candidates found (`MaxKmet/idea-validation-agents`, 392★, MIT; `ferdinandobons/startup-skill`, 540★, MIT; `garylab/MakeMoneyWithAI`, 608★, an unlicensed curated list, not a skill) are all far below the evidence bar this workspace already meets with `biz-council`'s sources (`last30days` 52,604★, `llm-council` 22,843★, `gstack` 122,570★), and their scope — idea validation, startup planning — already substantially overlaps what `biz-council` does. Route "how do I make money with AI" requests to `biz-council` instead of building a separate skill for it. **"Senior-focused money-making information" (income guidance specifically for elderly users) was also evaluated (2026-07-18) and explicitly deferred** — no GitHub candidate targets this niche specifically; the closest general match (`himself65/finance-skills`, 3,027★, MIT) is financial-analysis tooling, not income-generation guidance, and not age-targeted. Elder-directed financial/income content is also a heightened-risk category (scam and predatory-advice exposure) that this workspace's existing `agent-builder` Financial-action safety guard was written for in the autonomous-agent context, not the advice-content context — a future build here would need its own evidence-gated safety design, not a reuse of that guard, and should not proceed without a real methodology source rather than general "make money" video content. **Literal ERP/CRM/SCM software-system integration was evaluated (2026-07-18, sourced from a request for "ERP, CRM, SCM, finance" business-program coverage) and explicitly deferred as a separate skill** — the one purpose-built candidate found, `cluster-software/agent-crm` ("headless CRM for Claude & Codex," a16z Speedrun-backed), has only 111★, far below this workspace's evidence bar, though it's named in `biz-ops`'s Attribution as an early-stage option if a request specifically needs live CRM data rather than deal/pricing strategy. No candidate at all was found for ERP or SCM software integration specifically (as opposed to the vendor-management/procurement *advisory* work `biz-ops` already covers via `alirezarezvani/claude-skills`, 22,733★, MIT). Re-search before building rather than defaulting to `agent-crm` on star count growth alone — check whether a stronger-evidenced alternative has emerged.
