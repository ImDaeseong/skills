# Attribution and evidence log

Every third-party project this workspace's skills reference, cite, or depend on at runtime — plus the evaluations that led to a skill being built, deferred, or left unchanged. Not vendored into this repo — methodology reference only, or a runtime dependency cloned on demand. Grouped by which skill(s) each entry informs; entries with no skill listed are workspace-wide (origin story, shared runtime, cross-cutting evaluations).

This is an evidence/audit trail (per this workspace's LAW 1 star/license-ranking rule), not a usage guide — for how to install and invoke the skills, see [README.md](README.md); for per-skill invocation detail, see [USAGE.md](USAGE.md).
